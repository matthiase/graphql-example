## Development Setup

To get started:

Check the following commands work:
```
$ git -v
$ docker -v
$ docker-compose
```

And run:
```
$ git clone git@github.com:matthiase/graphql-example.git
$ cd graphql-example
$ docker-compose up --build
```

Now point your browser to http://localhost:8000 to start exploring the GraphQL
